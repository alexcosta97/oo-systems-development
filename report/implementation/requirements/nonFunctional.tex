\section{Non Functional}

\begin{itemize} % or {enumerate} for numbered list
  \item The login operation shouldn't take more than 2 seconds to be completed.
  \item The search for information shouldn't take more than 5 seconds
  \item New events should be advertised and appear automatically on the platform only two weeks before the start date
  \item The login for an agent should start working automatically on the contract start date.
  \item The login for an agent should stop working automatically after the contract end date.
  \item After any changes to the data of the system, it should automatically be updated to the system and changes should be noticed in real-time.
\end{itemize}
